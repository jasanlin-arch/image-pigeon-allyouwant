# 貼圖小鴿手

`Version` **1.7.0**

`What's New`
- 新增自動混合排版：同一份 Word 可同時包含直式與橫式圖片，自動依圖片方向排版
- 新增插入空白：可在任意位置插入空白佔位圖
- 合併圖片後可復原，不再是不可逆操作
- 修正 Windows 啟動閃退（emoji 字元造成的編碼錯誤）

### 📌 這是什麼

現場照片、對話截圖、轉帳明細、監視器畫面......

你是否因為如何整理這些圖片而感到苦惱呢？

放心！這個小工具就是來解決你的困擾的！


### ⭐ 特色功能
- 自動帶入備註
- 自動壓縮圖片
- 自動分割長截圖
- 排序、旋轉、合併圖片（支援復原）
- 插入空白佔位圖
- 儲存Word/壓縮過的圖片/專用檔案
- Word 排版支援自動混合直式與橫式圖片

### 📢 推薦技巧
#### ✅ 我想快速製作黏貼表
1. 編輯預設備註，匯入圖片
2. 修改個別備註、排序照片
3. 輸出成 Word 檔，完成！

#### ✅ 我想在同一份 Word 混合直式與橫式圖片
1. 匯入所有圖片（直式、橫式混合皆可）
2. 排版選擇「**自動混合（依圖片方向自動排佈）**」
3. 輸出 Word，程式會自動將直式圖片左右並排、橫式圖片上下排佈！

#### ✅ 我想整理照片
1. 匯入所有圖片
2. 使用「排序模式」快速排序圖片
3. 輸出圖片，就能得到壓縮且有序的圖片！

#### ✅ 我想保留編輯紀錄
1. 匯入圖片，編輯及排序
2. 另存成「貼圖小鴿手」專用檔案
3. 下次即可「讀取舊檔」，繼續編輯！
